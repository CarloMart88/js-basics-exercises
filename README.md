# JavaScript Snacks

Raccolta di 10 esercizi JavaScript risolti, utili per allenare:

- uso di map, filter, reduce
- manipolazione di array e stringhe
- funzioni pure
- logica di base

## Contenuto

Tutti gli esercizi si trovano nel file `snacks.js`.

## Esercizi inclusi

1. Somma dei numeri
2. Filtra numeri pari
3. Trova il massimo
4. Capitalizza parole
5. Lunghezza parole
6. Ordina oggetti per età
7. Rimuovi duplicati
8. Conta occorrenze di una lettera
9. Aggiungi simbolo €
10. …e altri piccoli esercizi di logica

## Come eseguire

```bash
node snacks.js
```
